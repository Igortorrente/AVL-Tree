# AVL-Tree

##About
 Generic Implementation of  AVL Tree in C

##Requirements
> * stdlib.h
> * stdbool.h
> * stdio.h
> * string.h

##Comparator

#####Model of Comparator
   ``` 
    int YourComparator(void* A, void* B )
   ```
#####Returns:
  ```
   0 if A = B
   1 if A > B
  -1 if A < B
  ```
